# 发布说明

## 当前版本

### v1.0.0 (2026)

#### 功能特性
- ✅ 任务管理：添加、编辑、删除任务
- 📊 优先级系统：支持 1-10 级优先级，可自定义颜色和文字
- ⏰ 倒计时提醒：文本格式倒计时（时：分：秒）
- 🎨 主题切换：亮色/暗色两种主题
- 📌 便签功能：置顶显示，支持拉伸和调整大小
- 🔔 系统托盘：最小化到托盘，双击恢复
- 🚀 单实例运行：防止重复启动
- 💾 数据持久化：自动保存任务和设置

#### 已知问题
- 无

#### 更新说明
- 修复了便签拉伸后文本换行失效的问题
- 修复了修改设置后便签位置重置的问题
- 优化了按钮颜色和样式
- 改进了暗色主题下的界面显示
- 实现了单实例运行功能

## 打包说明

### 开发环境运行
```bash
pip install -r requirements.txt
python memo_app.py
```

### 打包成可执行文件
```bash
python build_exe.py
```

打包完成后，可执行文件为 `备忘录.exe`
